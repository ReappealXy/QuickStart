一、软件概述
----------------------------------------------------------------

QuickStart 是一款面向个人的桌面快捷效率工具，集成记录、清单、翻译、
AI 对话等功能。本软件为纯本地化应用，旨在帮助用户快速记录想法、管理
日常任务和获取 AI 辅助。

三、数据隐私声明
----------------------------------------------------------------

1. 本地存储
   QuickStart 的所有数据（包括记录、清单、会话历史、配置等）均存储
   在用户本地计算机中，不会上传至任何第三方服务器。

2. API Key 安全
   您在设置中填写的 AI 服务 API Key 将使用 AES-256-GCM 加密后保存
   在本地 config.json 文件中。API Key 仅用于向您指定的 AI 服务商
   （如 DeepSeek）发送请求，不会被发送至其他任何地址。

3. 网络请求
   本软件仅在以下场景发起网络请求：
   - 调用您配置的 AI 服务 API（对话、翻译功能）
   除此之外，本软件不进行任何数据采集、统计或遥测。
